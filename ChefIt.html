CHEF IT!

Rogue like action adventure game.
Made with Unity.
Team Size: 4 (1 Engineer)

  • Complete inventory/equipment system. Designed with the designer in mind. Easy to add new items, equipment types, damage modifiers.
  • Procedural generation. Each "floor" in the forest is procedurally generated.
  • In-depth combat statistics.
  • Advanced Random Distribution System. This system is built to handle all kinds of cases. Used in the procedural generation for tile types and enemy horde spawns. Used in loot dropped from enemies. Ability to add loot tables within loot tables and modifiers such as "always drop" and "unique".
